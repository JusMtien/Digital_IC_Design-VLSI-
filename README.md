# Digital IC Design 
>  Weekly course projects and lab assignments for the CE222 course.
---

## 🛠️ Weakly Projects 
> The final report summarizes all the weekly projects from the course. First three weekly project is entry level for student to familiarize using tool to layout and simulation.


### Pmos_Nmos_Vsource_and_Noise_Margin
* Pmos and Nmos DC analysis 
* Analyse Pmos and Nmos Vsource 
* Find noise margin of Inverter

### Pmos_Nmos_size_and_delay
* Optimize NMOS and PMOS sizing based on saturation current (Ids) 
* Design schematic and layout for NMOS and PMOS devices
* Perform post-layout simulation to check propagation delay

### RC_Model_Delay
* Calculate Trise and Tfall manually using RC model formulas 
* Setup and run Hspice simulation for equivalent RC circuits
* Measure transition times and evaluate delay

### Inverter_Ring_Oscillator
* Evaluate Propagation Delay of an Inverter with Fanout 1 
* Extract RC parameters and calculate delay based on theoretical RC model 
* Design and simulate a 10GHz Ring Oscillator

### Decoder_2_to_4_Optimise_Design
* Survey and implement basic 2-to-4 Decoder architecture
* Optimize delay for 2-to-4 Decoder using Logical Effort method 
* Compute path effort, estimate stages, and find optimal gate size
